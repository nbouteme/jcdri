# jcdri
User-mode Nintendo Switch® Joy-Con© Driver for GNU/Linux

# Dependencies

- A decent C++ compiler
- libevdev
- hidapi
- libudev 
- cmake >= 3.9
- make
- pkg-config

## Building

From the root of the repository: 

`mkdir build && cd build && cmake .. && make`
